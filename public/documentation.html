Documentation Page

This web application will provide users with access to information about different movies, actors, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Get data about a single movie by title /movies/:Title GET None A JSON object holding data about a single movie, containing a description, genre, director, and image URL.
Example:
{
"_id" : ObjectId("5fce6c3f6f4a70ea53293d7d"),
"Title" : "Get Out",
"Description" : "Short description of movie.",
"Genre" : {
    "Name" : "Thriller",
    "Description" : "Short description of genre."
  },
"Director" : {
    "Name" : "Jordan Peele",
    "Bio" : "Short bio about director.",
    "Birth" : "1979"
  },
"ImagePath" : "link to image",
"Featured" : true
}
Get data about a genre by name/title /movies/genres/:Name GET None A text message describing the genre of the movie.
Example: "Horror films seek to elicit fear in its audience for entertainment purposes. They aim to evoke nightmares, fears, revulsions, and terror of the unknown and macabre."
Get data about a director by name /movies/directors/:Name GET None A JSON object holding data about a director, containing a bio, birth year, and death year(if applicable).
Example:
{
"Director" : {
    "Name" : "Jordan Peele",
    "Bio" : "Short bio about director.",
    "Birth" : "1979"
  }
}
Allow new users to register /users POST A JSON object holding data about the user, containing a username, password, email, and date of birth.
Example:
{
Username: "Rachael Rigby",
Password: "abcd1234!",
Email: "rrachaelar@gmail.com",
Birthday: "08/31/1993"
}		 A JSON object holding data about the user that was added.
Example:
{
Username: "Rachael Rigby",
Password: "abcd1234!",
Email: "rrachaelar@gmail.com",
Birthday: "08/31/1993"
}
Allow users to update their user info /users/:Username PUT A JSON object holding data about the user, containing a username, password, email, and date of birth, including the data to be updated.
Example:
{
Password: "newpassword",
Email: "newemail@aol.com",
}		 A text message indicating that the information was updated.
Example: "User information updated successfully!"
Allow users to add a movie to their list of favorites /users/:Username/Movies/:MovieID POST None A text message indicating that the movie was added to their list of favorites.
Example: "Movie successfully added to list of favorites!"
Allow users to remove a movie from their list of favorites /users/:Username/Movies/MovieID DELETE None A text message indicating that the movie was removed from their list of favorites.
Example: "Movie successfully removed from list of favorites."
Allow existing users to deregister /users/:Username DELETE None A text message indicating that the user successfully deregistered.
Example: "User successfully deregistered."
